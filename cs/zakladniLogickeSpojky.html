    Popis pravidel pro spojky:
  • Negace (¬ ) obrací hodnotu. Z 0 udělá 1 a obráceně.
  • Konjunkce (∧ ) je vždy lež, když se objeví jakákoli 0.
  • Disjunkce (∨ ) je vždy pravda, když se objeví jakákoli 1.
  • Implikace (⇒ ) když začíná 0, tak je automaticky pravda. Jen v případě 1=>0 je 0.
  • Ekvivalence (⇔ ) když oba nejsou stejné, tak je to 0.
    • Není umožněno pouštění více animací najednou.
