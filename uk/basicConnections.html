    Rules for connections:
  • Negation (¬ ) reverse value. From 0 create 1 and oposite.
  • Conjunction (∧ ) if found 0, than result always are 0.
  • Disjunction (∨ ) if found 1, than result always are 1.
  • Implication (⇒ ) when it begins with 0, then it is automatically true. Only in case of 1=>0 result is 0.
  • Equivalence (⇔ ) if both aren't same value, result is 0.
